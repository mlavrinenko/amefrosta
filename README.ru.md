Офлайн PWA-помощник для настольной игры **Послание из космоса** (Crowd Games).
Заменяет шифровальную ширму, терминал, планшеты для заметок, кубик и карты передач
удобным телефонным приложением — никаких физических компонентов не нужно, кроме
вашей карты сообщения. Игроки обмениваются зашифрованными словами через QR-коды
прямо за столом.

**Неофициальное фанатское приложение.** Не связано с издателем.

## Как играть

1. **Настройки** · Выберите язык игры и сторону (Инопланетянин или Учёный)
   во вкладке ⚙.
2. **Сообщение** · Бросьте кубик и впишите 3 секретных слова со своей карты
   сообщения.
3. **Шифр** · Инопланетянин: нажмите на полосу шифра, чтобы создать случайный
   допустимый шифр. Учёный: собирайте гипотезу буква за буквой по мере
   поступления сигналов.
4. **Передачи** · Введите слово передачи. Инопланетянин автоматически вычисляет
   счёт по своему шифру. Обменивайтесь записями через QR — Учёный показывает
   слово, Инопланетянин сканирует и возвращает счёт, Учёный сканирует обратно.
5. **Терминал** · Отмечайте роль каждой буквы (Доверие / Усиление / Подозрение).
   Нажатие — исключить, долгое нажатие — подтвердить. Подсказки предлагают
   гарантированные выводы.
6. **Заметки** · Свободный блокнот для любых записей.

Все данные хранятся на вашем устройстве. Никаких аккаунтов, серверов и сети —
только QR-коды лицом к лицу за столом.
