Офлайн PWA-помощник для настольной игры **Послание из космоса** (Crowd Games).
Заменяет шифровальную ширму, терминал, планшеты для заметок, кубик и карты передач
удобным телефонным приложением — никаких физических компонентов не нужно, кроме
вашей карты сообщения. Игроки обмениваются зашифрованными словами через QR-коды
прямо за столом.

**Неофициальное фанатское приложение.** Не связано с издателем.

## Как играть

1. **Настройки** · Выберите язык игры и сторону (Инопланетянин или Учёный)
   кнопкой ⚙ (шестерёнка).
2. **Сообщение** · Бросьте кубик и впишите 3 секретных слова со своей карты
   сообщения.
3. **Шифр** · Откройте полосу шифра в шапке.
   - Инопланетянин: нажмите «Сгенерировать» для случайного допустимого шифра или
     заполните слоты вручную — выбор предлагает только буквы, подходящие ярусу.
   - Учёный: собирайте гипотезу буква за буквой; завершённые попытки уходят в
     архив, ранжируются по числу совпавших сигналов и помечаются как недопустимые.
4. **Передачи** · Введите слово передачи. Инопланетянин автоматически вычисляет
   счёт по своему шифру.
   - Живой предпросмотр разбивает слово на +Доверие, ×Усиление и −Подозрение
     и отмечает уже использованные буквы.
   - Обменивайтесь записями через QR — Учёный показывает слово, Инопланетянин
     сканирует и возвращает счёт, Учёный сканирует обратно (или вводит вручную).
5. **Терминал** · Отмечайте роль каждой буквы (Доверие / Усиление / Подозрение).
   Нажатие — исключить, долгое нажатие — подтвердить.
   - Меняйте порядок строк: А–Я, использованные вперёд или по частоте.
   - Подсказки указывают на гарантированные выводы и применяются одним нажатием.
6. **Заметки** · Свободный блокнот, открывается из шапки, — для любых записей.

Все данные хранятся на вашем устройстве. Никаких аккаунтов, серверов и сети —
только QR-коды лицом к лицу за столом.
